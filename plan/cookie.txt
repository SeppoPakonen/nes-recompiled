## Quick Reference

### Build Commands
```bash
# Build recompiler
cmake -G Ninja -B build .
ninja -C build

# Recompile ROM
./build/bin/nesrecomp roms/a.nes -o output/a

# Build generated project
cmake -G Ninja -S output/a -B output/a/build
ninja -C output/a/build

# Run game
./output/a/build/a
```

### Key Directories
- `recompiler/` - Recompiler source
- `runtime/` - Runtime library (nesrt)
- `roms/` - ROM files
- `output/` - Generated C projects
- `plan/` - This plan directory

### Key Files
- `recompiler/include/recompiler/rom.h` - ROM loader
- `recompiler/include/recompiler/decoder.h` - CPU decoder
- `recompiler/include/recompiler/analyzer.h` - Control flow
- `recompiler/include/recompiler/ir/ir.h` - IR definitions
- `recompiler/include/recompiler/codegen/c_emitter.h` - C codegen
- `runtime/include/nesrt.h` - Main runtime header
- `runtime/include/gbrt.h` - OLD GB runtime (to replace)
