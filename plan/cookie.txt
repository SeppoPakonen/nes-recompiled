## Quick Reference

### Build Commands
```bash
# Build recompiler
cmake -G Ninja -B build .
ninja -C build

# Recompile ROM
./build/bin/nesrecomp roms/a.nes -o output/a

# Build generated project
cmake -G Ninja -S output/a -B output/a/build
ninja -C output/a/build

# Run game
./output/a/build/a
```

### Key Directories
- `recompiler/` - Recompiler source
- `runtime/` - Runtime library (nesrt)
- `roms/` - ROM files
- `output/` - Generated C projects
